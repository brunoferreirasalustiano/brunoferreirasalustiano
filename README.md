## 🧩 Projetos em destaque

### 🛒 Meu Carrinho de Compras
Aplicativo mobile em **React Native + TypeScript** para lista de compras, controle de orçamento, cálculo de total, saldo restante, persistência local e suporte a múltiplos idiomas.

**Tecnologias:** React Native, TypeScript, Expo, Context API, AsyncStorage, EAS Build  
**Repositório:** https://github.com/brunoferreirasalustiano/meu-Carrinho-de-Compras

---

### 📄 Orçamento Rápido
Aplicativo mobile premium em **React Native + Expo** para criação de orçamentos profissionais, histórico local, geração de PDF e compartilhamento com clientes por WhatsApp e outros canais.

**Tecnologias:** React Native, Expo, TypeScript, AsyncStorage, PDF, EAS Build  
**Repositório:** https://github.com/brunoferreirasalustiano/APP-ORCAMENTO-RAPIDO

---

### 📊 Monitor Universal de APIs
Projeto full stack para monitoramento de latência, consumo e disponibilidade de APIs, com dashboard, backend Node.js, banco PostgreSQL/Supabase e exportação de relatórios.

**Tecnologias:** Node.js, Express, PostgreSQL, Supabase, JavaScript, Chart.js  
**Back-end:** https://github.com/brunoferreirasalustiano/Meu-monitor-api  
**Front-end:** https://github.com/brunoferreirasalustiano/monitor-api-frontend

---

### 🔎 Consulta CNPJ
Sistema web para consulta de CNPJs brasileiros, com cache local, tratamento de dados cadastrais, feedback de carregamento e interface responsiva.

**Tecnologias:** HTML5, CSS3, JavaScript, LocalStorage, API pública  
**Repositório:** https://github.com/brunoferreirasalustiano/consulta-cnpj  
**Demo:** https://brunoferreirasalustiano.github.io/consulta-cnpj/

---

### 💼 Portfólio Full Stack
Portfólio profissional com arquitetura desacoplada, front-end hospedado no GitHub Pages, backend com API própria e persistência em Supabase/PostgreSQL.

**Tecnologias:** HTML, CSS, JavaScript, Node.js, Supabase, PostgreSQL, Resend API  
**Repositório:** https://github.com/brunoferreirasalustiano/6-Portifolio-Bruno
